CONSTANTS.PY

Файл, который необходимо обновлять, чтобы получать обновленную информацию.
Содержит в себе пути до EXCEL/CSV файлов c данными.


CHECK_COMBINER.PY

Для того чтобы взять чеки из папки с данными и обработать с выдачей файла по кол-ву чеков на АЗС в определенный день.
Есть функция с группировкой по часам.


NEW_CHECK_ADD.PY

Для того чтобы брать текущий файл с чеками и добавлять туда новые данные.
Функция собрана только для обработки данных по дням (не по часам).


SCRIPT.PY

Запускает файл с обновлением массива чеков. Получаешь информацию на выходе в виде нового CSV.
При этом предыдущий файл остается неизменным.


Инструкция:
1. Скачать файл с чеками ТРК по ссылке: https://ptrbi.lukoil.com/DataExtractor/view/%D0%A7%D0%B5%D0%BA%D0%B8%20%D1%81%20%D0%A2%D0%A0%D0%9A.ini
2. Добавить файл в папку "Чеки"
3.1. Если нет исходного массива с обработанными чеками - запустить скрипт в CHECK_COMBINER.PY и ожидать. Предварительно задать пути в CONSTANTS.PY.

3.2.1. Если имеется файл с уже обработанными чеками и необходимо присоединить новый обработанный файл - один файл, добавленный последним.
- Задать пути до папок и файлов в CONSTANTS.PY.
- В качестве mode выбрать 'latest' - он выберет последний добавленный файл и присоединит.

3.2.2. Если имеется файл с уже обработанными чеками и необходимо присоединить группу обработанных файлов.
- Задать пути до папок и файлов в CONSTANTS.PY.
- Передать в mode - 'manual'
- В значение files передать список из путей до файлов, необходимых к добавлению.

3.2.3. Если нет обработанного файл и хочешь вызвать функцию аналогичную пункту 3.1. для создания массива с нуля.
- Задать пути до папок и файлов в CONSTANTS.PY.
- Передать в mode - 'state'
- Автоматически запустится скрипт создания массива из всех файлов, лежащих в папке "Чеки"
